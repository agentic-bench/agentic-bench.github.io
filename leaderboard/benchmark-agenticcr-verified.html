Displayed Primary Metric: Code Review Capability (Harmonic Mean)

The leaderboard displays agents ranked by Code Review Capability, calculated as the harmonic mean of localization accuracy and human alignment. This metric encourages balanced performance: agents must excel at BOTH finding the right location AND providing aligned feedback. The harmonic mean penalizes imbalance more than arithmetic mean, ensuring neither aspect is neglected.

Formula:
Code Review Capability = 2 / (1/alignment + 1/localization)

where:
  alignment = is_llm_human_aligned (% aligned among localized comments)
  localization = is_human_llm_location_matched (% comments within ±5 lines of GT)
Why Harmonic Mean?
  • Penalizes Imbalance: An agent with 80% alignment + 20% localization scores 32% (vs 50% arithmetic mean)
  • Both Required: Code review needs both correct location AND aligned feedback
  • Standard Practice: Similar to F1-score (harmonic mean of precision & recall)

Background Analysis: Signal-to-Noise Ratio (SNR)

Behind the scenes, agents are also scored by Signal-to-Noise Ratio (SNR), a metric that penalizes high-volume generators and rewards efficiency. This metric is stored in the data but hidden from the leaderboard view to keep the interface focused. SNR measures how efficiently an agent generates useful (aligned) vs useless (non-localized or misaligned) comments.

SNR Formula:
SNR = A / (T − A)

where:
  A = number of aligned comments (signal)
  T = total generated comments
  (T − A) = total noise (misaligned + non-localized)
Why SNR Matters:
  • Penalizes Volume: 1000 comments with 50 aligned (SNR=0.053) ranks lower than 200 comments with 30 aligned (SNR=0.176)
  • Rewards Efficiency: Encourages agents to be selective rather than spammy
  • Addresses Real Problem: High performers can generate massive noise alongside aligned comments
Available for Analysis: SNR values are stored in the leaderboard data under overall_weighted_score for agents and research purposes. Use src/extra/snr_analysis.py to compute detailed SNR comparisons.
