About Evaluation Versioning

Each benchmark has an evaluation version that represents the entire set of metrics and LLM judges used. When evaluators change (new metrics, updated prompts, or different LLM models), the version is incremented.

  • Major version bump (X.0): Add/remove evaluator, change LLM model, major prompt changes
  • Minor version bump (x.Y): Model version update, bug fixes, minor improvements

Note: Results from different evaluation versions may not be directly comparable. Always check which version was used when comparing agent performance.
