SCRBench

Bug Detection

Benchmarks whether an LLM code reviewer can identify immature vulnerabilities in pull requests. Each diff contains a vulnerability-introducing changes, the earliest known changes that led to real vulnerabilities (CVEs). The ground truth specifies the exact file, line range, CWE classification, and bug description. A successful agent must identify and flag the bug at the correct location.

Example 1 — SaltStack Path Traversal (CVE-2017-14695)

0976f8f7131975a1ae29b2724069a301a870a46d CWE-22 View Commit ↗
Bug: Directory traversal vulnerability in minion id validation in SaltStack Salt. Remote minions with incorrect credentials could authenticate to a master via a crafted minion ID (CVE-2017-14695 — incomplete fix for CVE-2017-12791).

Diff

--- a/salt/utils/verify.py
+++ b/salt/utils/verify.py
@@ -484,7 +484,7 @@ def clean_id(id_):
     '''
     Returns if the passed id is clean.
     '''
-    if re.search(r'\.\.\{sep}'.format(sep=os.sep), id_):
+    if re.search(r'\.\.\{sep}'.format(sep=os.sep), id_):
         return False
     return True

Ground Truth

CWE-22 salt/utils/verify.py : line 487
Path Traversal — the regex r'\.\.\{sep}' does not correctly escape the separator, allowing crafted minion IDs with ../ sequences to bypass validation.

bf9c4ba9e81fd066b4d133eaddf2da2c6b3005aa CWE-19 View Commit ↗
Bug: modules/chef.py in SaltStack before 2014.7.4 does not properly handle files in /tmp, allowing local attackers to conduct symlink attacks (CVE-2014-3563).

Diff

--- a/salt/modules/chef.py
+++ b/salt/modules/chef.py

-    tmp_file = tempfile.mktemp()
+    tmp_file = tempfile.mkstemp()[1]
     if api:
         chef_cmd.append('chef-client --format=json --log_level=fatal')
     else:

Ground Truth

CWE-19 salt/modules/chef.py : lines 26–46
tempfile.mktemp() creates a temporary filename without opening the file, leaving a race condition window where an attacker can create a symlink at that path. Use tempfile.mkstemp() instead to atomically create and open the file.
