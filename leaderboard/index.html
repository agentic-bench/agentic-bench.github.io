    How to Submit

    Evaluate your agent on any benchmark and submit results via a pull request. All submissions must follow the unified llm-comments format ↗.

    1. Run your agent on the benchmark dataset:
      benchmarks/{benchmark}/input-dataset/dataset.jsonl
    2. Format output — reformat into the unified llm-comments format and save as:
      benchmarks/{benchmark}/llm-comments/{agent_id}_{YYYYMMDD-HHMM}.jsonl
    3. Evaluate
      python src/evaluator.py --benchmark {benchmark} --input path/to/your-file.jsonl
    4. Update leaderboard
      python src/leaderboard.py
    5. Open a PR including:
      llm-comments/your-file.jsonl, eval-results/*_comments.jsonl, eval-results/*_trajectory.jsonl, and updated leaderboard/data/ files.
